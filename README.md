# Prédire la date de sortie d'une chanson

L’objectif de ce projet est de développer un programme capable de prédire l’année de sortie d’une chanson. Pour ce faire, nous disposons de 4578 exemples de chansons dont on connaît la date de sortie et 90 descripteurs pour chaque exemple de chansons.

Ainsi, nous avons développé des algorithmes que nous entraînerons sur ces données. Par la suite, nous testons ces algorithmes sur 2289 exemples de chansons dont nous ne connaissons pas les dates sorties et ainsi les prédire.

Pour réaliser ce projet, nous avons tout d’abord importer les données et les visualiser. Dans un second temps, nous avons appliquer des algorithmes élémentaires pour répondre au problème de classification des chansons et visualiser les performances de chacune de ces méthodes et leur temps de calcul. 
Ensuite, nous avons utilisé des algorithmes de régularisation pour éviter le phénomène de sur-apprentissage.
En fin, nous avons estimer les hyperparamètres pour chacun de ces algorithmes par validation croisée, sélectionner le meilleur algorithme et faire la prédiction des données non-étiquetées.
